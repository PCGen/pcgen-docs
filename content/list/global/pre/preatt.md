## Status

None

## Syntax

`PREATT:x`

## Parameters

-   x: Number (Base Attack Bonus number).



What it does
------------

Number indicates the minimum base attack bonus.

Highest PREATT possible is 20.

Example
-------

`PREATT:6`

This would apply only if the PC had minimum base attack bonus of 6.

## Status

None

## Syntax

`PRECITY:x`

## Parameters

-   x: Text (The name of a city that you must live in).



What it does
------------

Sets the required city. A character's city is set in the [Description
Tab](/tab/description.html)

Example
-------

`PRECITY:Klamath`

Character must currently reside in "Klamath".

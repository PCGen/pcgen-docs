## Status

None

## Syntax

`PRESPELL:x,y,y`

## Parameters

-   x: Number (The number of spells required).
-   y: Text (The name of a spell).



What it does
------------

Sets spell requirements.

Example
-------

`PRESPELL:1,Magic Missile,Lightning Bolt`

Character must have either Magic Missile OR Lightning Bolt in their
spell list.

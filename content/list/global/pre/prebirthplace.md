## Status

None

## Syntax

`PREBIRTHPLACE:x`

## Parameters

-   x: Text (A birthplace name).



What it does
------------

Character's birthplace must match the listed text.

Example
-------

`PREBIRTHPLACE:Klamath`

Character must have been born in "Klamath".

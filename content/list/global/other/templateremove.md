## Status

None

## Syntax

`TEMPLATE:x.REMOVE`

## Parameters

-   x: Text (Template name)



What it does
------------

This tag removes the designated template.

Example
-------

`TEMPLATE:Lycanthrope.REMOVE`

Removes the "Lycanthrope" template..

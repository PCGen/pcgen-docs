## Status

None

## Syntax

`DEITY:x`

## Parameters

-   x: Text



What it does
------------

The list that the character may follow, it is | (pipe) delimited and
defaults to ANY if not present

Examples
--------

`DEITY:Zeus`

Class may only have "Zeus" as its Deity.

`DEITY:Zeus|Athena`

Class may have "Zeus" or "Athena" as its Deity.

Where it is used
----------------

Class Line.

## Status

None

## Syntax

`HD:x`

## Parameters

-   x: Number (4, 6, 8, 10, 12. default is 0)



What it does
------------

This is the Hit Die used to determine hit points at each level gained.

Example
-------

`HD:10`

The hit dice per level is 10.

Where it is used
----------------

Class Line.

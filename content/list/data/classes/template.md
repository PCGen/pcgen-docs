## Status

None

## Syntax

`TEMPLATE:x|x`

## Parameters

-   x: Text (Template Name)



What it does
------------

This is a pipe-delimited (|) list of templates that are granted to the
class.

Example
-------

`TEMPLATE:Incorporeal|Undead|Celestial`

The class uses the "Incorporeal", "Undead" and "Celestial" templates.

Where it is used
----------------

Class Level Line

## Syntax

`XTRAFEATS:x`

## Parameters

-   x: Number



What it does
------------

This is used to determine how many extra feats the class will get at
first level only.

Example
-------

`XTRAFEATS:1`

The class will gets 1 extra feat.

Where it is used
----------------

Class Line.

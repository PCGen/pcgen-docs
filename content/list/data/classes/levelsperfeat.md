## Status

None

## Syntax

`LEVELSPERFEAT:x`

## Parameters

-   x: Number



What it does
------------

This is the number of levels it takes for a monster to gain a new feat.

Example
-------

`LEVELSPERFEAT:2`

The class gets 1 feat per 2 levels.

Where it is used
----------------

Class Line.

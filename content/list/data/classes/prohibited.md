## Status

None

## Syntax

`PROHIBITED:x,x`

## Parameters

-   Variables Used: Text



What it does
------------

This is a comma delimited list of spell schools that are prohibited to
the class (no spells from that school can be taken).

Example
-------

`PROHIBITED:Divination,Necromancy`

The class cannot use Divination or Necromancy.

Where it is used
----------------

Class Line.

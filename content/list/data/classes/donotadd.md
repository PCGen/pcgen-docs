## Status

Added 5.17.x

## Syntax

`DONOTADD:x|x`

## Parameters

-   x: HITDIE
-   x: SKILLPOINTS



What it does
------------

Suppresses the hit die and/or skill points for the associated level.

Example
-------

`5 <tab> DONOTADD:HITDIE`

The character will not receive an additional hit die when reaching fifth
level.

Where it is used
----------------

Class Level Line.

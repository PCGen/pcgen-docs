## Status

None

## Syntax

`BASEQTY:x`

## Parameters

-   x: Number (standard bundle quantity)



What it does
------------

The numeric value supplied is the number of items in this bundle. ie.
Arrows (20) should have.

Example
-------

`BASEQTY:20`

20 items in a bundle (e.g. Arrows).

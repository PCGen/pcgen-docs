## Status

None

## Syntax

`ACCHECK:x`

## Parameters

-   x: Number (Armor Check Penalty)



What it does
------------

The numeric value supplied is the Armor Check penalty applied to certain
skills that have an Armor Check.

Example
-------

`ACCHECK:-6`

Item has an armor check penalty of -6.

## Status

None

## Syntax

`COSTPRE:x`

## Parameters

-   x: Number (Cost before resizing)



What it does
------------

The cost applied to the equipment before resizing an item occurs.

Example
-------

`COSTPRE:100`

Item cost is 100 gold before resizing.

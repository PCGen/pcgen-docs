## Status

None

## Syntax

`ARMORTYPE:x|y`

## Parameters

-   x: Text (Current armor TYPE)
-   y: Text (New armor TYPE)



What it does
------------

This will change the armor type and is | (pipe) delimited.

Example
-------

`ARMORTYPE:Medium|Heavy`

Changes the armor type from "Medium" to "Heavy".

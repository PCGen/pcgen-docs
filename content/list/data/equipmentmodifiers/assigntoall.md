## Status

None

## Syntax

`ASSIGNTOALL:x`

## Parameters

-   x: Boolean (YES)



What it does
------------

This is for double weapons only. If this tag is present then the
modifiers will be applied to both heads.

Example
-------

`ASSIGNTOALL:YES`

Assigns modifiers to both heads.

**Default Value:**

`NO`

## Status

None

## Syntax

`CHARGES:x|y`

## Parameters

-   x: Number (Minimum number of charges)
-   y: Number (Maximum number of charges)



What it does
------------

Sets the charges for a magic item.

Example
-------

`CHARGES:1|50`

Sets the charges for a magic item at a minimum of 1 and maximum of 50.

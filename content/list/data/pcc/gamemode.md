## Status

None

## Syntax

`GAMEMODE:x`

## Parameters

-   x: Text (Name of GameMode)



What it does
------------

-   This *MUST* be 2nd in the file.
-   This tells PCGen what genre this is in PCGen and must be entered
    exactly as shown.
-   The `GAMEMODE` tag is case-sensitive.

Examples
--------

`GAMEMODE:3e`

This PCC file uses the Third Edition SRD gamemode.

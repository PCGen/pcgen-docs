## Status

None

## Syntax

`SOURCEWEB:x`

## Parameters

-   x: Text (Web address)



<span id="sourceweb"></span> \*\*\* Updated 4.3.10

What it does
------------

The URL that points directly to this product/book.

Example
-------

`SOURCEWEB:http://www.foogames.com/product.php?products=654321`

The web site for this source is
"SOURCEWEB:http://www.foogames.com/product.php?products=654321".

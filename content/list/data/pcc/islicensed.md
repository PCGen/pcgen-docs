## Status

None

## Syntax

`ISLICENSED:x`

## Parameters

-   x: Boolean (YES or NO)



<span id="islicensed"></span> \*\*\* New 5.3.3

What it does
------------

Used to indicate if sources have a special license, and activated the
LICENSE tag, which will display information about the license to the
end-user.

Examples
--------

`ISLICENSED:YES`

`ISLICENSED:NO`

**Default Value:**

`NO`

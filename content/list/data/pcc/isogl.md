## Status

None

## Syntax

`ISOGL:x`

## Parameters

-   x: Boolean (YES or NO)



What it does
------------

Used to indicate if sources are OGL compliant. If `ISOGL:YES` then its
COPYRIGHT info gets inserted into the OGL license. This tag is optional.

Examples
--------

`ISOGL:YES`

`ISOGL:NO`

**Default Value:**

`NO`

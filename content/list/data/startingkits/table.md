## Status

None

## Syntax

`TABLE:x`

## Parameters

-   x: Text (Table name)



What it does
------------

Identifies a table of values to be used in conjunction with the `VALUES`
tag and called by a `LOOKUP` tag.

Example
-------

`TABLE:Minor Special Ability (P/S)`

Would create a table called "Minor Special Ability (P/S)".

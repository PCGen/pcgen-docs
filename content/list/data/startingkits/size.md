## Status

New 5.9.2

## Syntax

`SIZE:x`

## Parameters

-   x: Size category (F,D,T,S,M,L,H,G,C)
-   x: PC (matches the equipment to the
    character's size)



What it does
------------

Enlarges or reduces the GEAR item to the size specified.

Example
-------

`SIZE:S`

The item is set to size small.

Where it is used
----------------

GEAR lines

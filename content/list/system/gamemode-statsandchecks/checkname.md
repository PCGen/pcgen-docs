## Status

None

## Syntax

`CHECKNAME:x`

## Parameters

-   x: Text (The name for the check you wish to use).



What it does
------------

Defines checks to later be used for saving throws.

Example
-------

`CHECKNAME:Fortitude`

Defines "Fortitude" as a valid check.

Where it is used
----------------

CHECKNAME Line.

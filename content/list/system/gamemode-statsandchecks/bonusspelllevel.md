## Status

None

## Syntax

`BONUSSPELLLEVEL:x`

## Parameters

-   x: Number (The level you are defining bonus
    spells for).



What it does
------------

Defines the level of bonus spells.

Example
-------

`BONUSSPELLLEVEL:1`

Defines that bonus spells are for level 1.

Where it is used
----------------

BONUSSPELLLEVEL Line.

## Status

None

## Syntax

`STATNAME:x`

## Parameters

-   x: Text (The full name of the stat).



What it does
------------

Used to create stats in the program for the character to build upon.

Example
-------

`STATNAME:Awareness`

Creates a Stat named "Awareness".

Where it is used
----------------

STATNAME Line.

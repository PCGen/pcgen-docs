## Status

None

## Syntax

`ABB:x`

## Parameters

-   x: Text (The abbreviation for the stat - usually
    three letters, capitalized).



What it does
------------

Defines the three letter Abbreviation for a statistic.

Example
-------

`ABB:AWR`

Example abbreviation of above example of "Awareness" stat.

Where it is used
----------------

STATNAME Line.

## Status

None

## Syntax

`DOWN:x,x`

## Parameters




**Varibles used:** (x) alpha numerical

What it does
------------

This comma (,) delimited value determines the next die value in an
incremental set

Example
-------

`DOWN:1d4,1d3,1d2,1`

The next die in the progression will be 1d4, then 1d3 if chosen again,
etc.

Where it is used
----------------

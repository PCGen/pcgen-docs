## Status

New 5.8.0

## Syntax

`BABABBREV:x`

## Parameters

-   x: Text (What the characters BAB should be called).



What it does
------------

Specifies the text to use for the abbreviation for Base Attack Bonus.

Example
-------

`BABABBREV:OB`

Displays OB: instead of BAB:.

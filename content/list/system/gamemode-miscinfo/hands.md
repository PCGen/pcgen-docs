## Status

New 5.5.2

## Syntax

`HANDS:x`

## Parameters

-   x: Number (minimum number of hands)



What it does
------------

Defines the minimum number of hands needed to wield the weapon for the
wield category.

Example
-------

`HANDS:2`

Weapon must be wielded with two hands.

Where it is used
----------------

After a WIELDCATEGORY tag.

## Status

New 5.17.0

## Syntax

`STARTSTATMIN:x`

## Parameters

-   x: Number (Minimum Stat).



What it does
------------

-   Defines the minimum starting stat for the Game Mode.
-   Defaults to 3 if it remains undefined in the Game Mode.

Example
-------

`STARTSTATMIN:5`

Sets the minimum STAT to "5".

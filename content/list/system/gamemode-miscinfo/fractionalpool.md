## Status

New 5.11.1

## Syntax

`FRACTIONALPOOL:x`

## Parameters

-   x: Boolean (Yes or No)



What it does
------------

Used in
[ABILITYCATEGORY](/list/system/gamemode-miscinfo/abilitycategory.html)
lines to define if the pool can use fractional amounts.

Example
-------

`FRACTIONALPOOL:No`

Define that the category's pool can only be a whole number.

Where it is used
----------------

Used in
[ABILITYCATEGORY](/list/system/gamemode-miscinfo/abilitycategory.html)
lines.

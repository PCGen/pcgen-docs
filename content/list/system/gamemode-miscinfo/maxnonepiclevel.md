## Status

New 6.03.x

## Syntax

`MAXNONEPICLEVEL:x`

## Parameters

-   x: Number (Integer)



What it does
------------

This sets the maximum non-epic levels allowed for the gamemode.

Example
-------

`MAXNONEPICLEVEL:20`

Levels above 20th are concidered epic levels.

## Status

None

## Syntax

`MENUTOOLTIP:x`

## Parameters

-   x: Text (Tooltip to be displayed for the
    previous MENUENTRY).



What it does
------------

Defines the tooltip displayed for the previously defined menuentry.

Example
-------

`MENUTOOLTIP:`

Defines a menu entry tooltip that will display as "Use 3e character
creation settings".

## Status

None

## Syntax

`MENUENTRY:x`

## Parameters

-   x: Text (Text you want displayed as a menu item
    under the settings/campaign menu).



What it does
------------

Defines menu entries under Settings-&gt;Campaign (Use & for mnemonic, &&
for &).

Example
-------

`&S&&M`

Defines a menu entry that will display as "S&M".

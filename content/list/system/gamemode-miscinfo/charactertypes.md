## Status

New 5.17.12

## Syntax

`CHARACTERTYPES:x|x`

## Parameters

-   x: Text (Character type, e.g. PC, NPC, etc.).



What it does
------------

Lists the character types that are avsilable in the gamemode.

Example
-------

CHARACTERTYPES:PC|NPC

The character types available on this gamemode are &QUOT;PC&QUOT; and
&QUOT;NPC&QUOT;.

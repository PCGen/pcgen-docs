## Status

None

## Syntax

`DISPLAYORDER:x`

## Parameters

-   x: Number (Order of priority)



What it does
------------

Used to determine order of display in the Game Mode/Campaign menu.

Example
-------

`DISPLAYORDER:1`

The Game Mode/Campaign menu will show this as the first Game Mode.

## Status

New 5.8.0

## Syntax

`WEAPONNONPROFPENALTY:x`

## Parameters

-   x: Number (The penalty to be applied)



What it does
------------

Defines the penalty to attack for not being proficient.

Example
-------

`WEAPONNONPROFPENALTY:-4`

Defines a -4 penalty to attack for not being proficient in a weapon.

## Status

New 5.9

## Syntax

`SQUARESIZE:x`

## Parameters

-   x: Number (in feet).



What it does
------------

Specifies the size of a battlemap square in feet

Example
-------

`SQUARESIZE:5`

Sets the size of a battlemap square to 5 feet.

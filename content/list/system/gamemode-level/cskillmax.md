## Status

None

## Syntax

`CSKILLMAX:x`

## Parameters

-   x: Number or Formula (Skill Rank)



What it does
------------

Sets the maximum rank for class skills.

Examples
--------

`CSKILLMAX:LEVEL+3`

The standard formula for maximum class skill ranks for the R/SRD.

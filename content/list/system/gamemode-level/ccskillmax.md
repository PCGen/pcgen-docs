## Status

None

## Syntax

`CCSKILLMAX:x`

## Parameters

-   x: Number or Formula (Skill Rank)



What it does
------------

Sets the maximum rank for cross class skills.

Examples
--------

`CCSKILLMAX:(LEVEL+3)/2`

The standard formula for maximum cross-class skill ranks for the R/SRD.

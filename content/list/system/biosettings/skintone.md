## Status

None

## Syntax

`SKINTONE:x|x`

## Parameters

-   x: Text (Skin tone color)



What it does
------------

The entries in this tag are used by the Description tabs randomizing
function for the skin tone for the associated race.

Example
-------

`SKINTONE:Tanned|Pasty|Gray`

Skin tone is Tanned Pasty or Gray.

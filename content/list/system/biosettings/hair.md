## Status

None

## Syntax

`HAIR:x|x`

## Parameters

-   x: Text (Hair Color)



What it does
------------

The entries in this tag are used by the Description tabs randomizing
function for the hair for the associated race.

Example
-------

`HAIR:Blue|Black|Brown|Gray`

Hair may be Blue, Black, Brown or Grey.

## Status

None

## Syntax

`BASEAGE:x`

## Parameters

-   x: Number (The minimum age for the associated age
    set and race)



What it does
------------

Sets the minimum age to be considered in the age set.

Example
-------

`BASEAGE:35`

Base Age is 35.

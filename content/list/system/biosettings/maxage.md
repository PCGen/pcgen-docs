## Status

None

## Syntax

`MAXAGE:x`

## Parameters

-   x: Number (The maximum age for the associated age
    set and race)



What it does
------------

Sets the maximum age to be considered in the age set.

Example
-------

`MAXAGE:50`

Maximum age for this set is 50.

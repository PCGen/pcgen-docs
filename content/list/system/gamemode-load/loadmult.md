## Status

None

## Syntax

`LOADMULT:x`

## Parameters

-   x: Number



What it does
------------

Set the original value that is changed in the <span class="lstfile">
SizeAdjustment.lst </span> file.&lt;.p&gt;

Example
-------

`LOADMULT:4`\

Sets the original value to 1.

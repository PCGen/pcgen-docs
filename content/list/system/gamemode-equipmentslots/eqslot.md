## Status

None

## Syntax

`EQSLOT:x`

## Parameters

-   x: Text (Name of slot)



What it does
------------

This is the first tag in an equipmentslot line and sets the name of the
slot. It is followed by a CONTAINS tag and a NUMBER tag.

Example
-------

`EQSLOT:Eyes`

Sets the name of this slot as Eyes.
